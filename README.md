# Gost-Utils
一款自动化Gost隧道的工具箱

### 使用方法
```shell
wget -N --no-check-certificate "https://gitee.com/bobkjl/Gost-Utils/raw/master/install.sh" && bash install.sh
```

### 功能说明
- 一键安装Gost
- 一键配置Gost服务端
- 一键配置Gost客户端
